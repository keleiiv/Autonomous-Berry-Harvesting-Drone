# AI-Based Autonomous Berry Harvesting Drone Using Computer Vision

## 🚧 Status: Active Development / Prototyping Phase
This repository serves as the central workspace for our major engineering project. The project is currently in the architectural design and software simulation phase.

---

## 📌 Project Overview
An autonomous aerial robotic system designed to locate, track, and harvest ripe crops using Edge AI, computer vision, and custom-designed end-effectors[cite: 1]. This project aims to bring industrial-grade automation standards to precision farming[cite: 1].

---

## 🛠️ Proposed Tech Stack & Architecture
* **Software & AI:** Python, OpenCV, Machine Learning, ROS 2[cite: 1]
* **Embedded Hardware:** Pixhawk Flight Controller, ESP32, Raspberry Pi[cite: 1]
* **Mechanical Design:** SolidWorks / Autodesk Fusion 360 (For CAD modeling and structural layout)[cite: 1]
* **Version Control:** Git & GitHub[cite: 1]
